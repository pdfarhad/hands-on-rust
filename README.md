# Hands on Rust

## Installing Rust

1. Linux or macOS
    - Run the following command in a terminal 
    
    `curl --proto '=https' --tlsv1.2 https://sh.rustup.rs -sSf | sh`
2. Windows 
    - See https://www.rust-lang.org/tools/install 
    - You'll also need the C++ build tools for VS 2013 or later
